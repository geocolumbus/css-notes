CSS Notes

Notes on the CSS language.

Flexbox

flex parent
-----------
flex-direction: column, column-reverse, row, row-reverse
flex-wrap: wrap, no-wrap, wrap-reverse
flex-flow: (flex-direction + flex-wrap) row wrap
justify-content: center, flex-start, flex-end, space-around, space-between
align-items: center, flex-start, flex-end, stretch, baseline
align-content: space-between, space-around, stretch, center, flex-start, flex-end

flex children
-------------
order: 1, 2, ... n
flex-grow: 0 (relative number, default 0)
flex-shrink: 1 (relative number, default 1)
flex-basis: 200px (initial length)
align-self: center, flex-start, flex-end, stretch, baseline (overrides align-items on parent)

responsive
----------
.flex-container {
  display: flex;
  flex-direction: row;
}

/* Responsive layout - makes a one column layout instead of a two-column layout */
@media (max-width: 800px) {
  .flex-container {
    flex-direction: column;
  }
}
